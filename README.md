# 🧮 Calculator

A sleek, cyberpunk-styled web calculator built with vanilla HTML, CSS, and JavaScript. Converted from a Python Tkinter desktop app to a fully browser-based web app.

## 🔗 Live Demo

👉 **[Click here to open the Calculator](https://puspendu-calculator.netlify.app/)**

> *(Replace the link above with your Netlify/GitHub Pages URL)*

---

## ✨ Features

- **Basic Mode** — Addition, subtraction, multiplication, division, percentage, parentheses
- **Scientific Mode** — sin, cos, tan, log, square root, x², π
- **Keyboard Support** — Type numbers and operators directly from your keyboard
- **Expression History** — See your last expression above the result
- **Error Handling** — Shake animation on invalid input instead of a popup
- **Responsive Design** — Works on desktop and mobile browsers

## 🛠️ Tech Stack

- HTML5
- CSS3 (animations, CSS variables, Grid layout)
- Vanilla JavaScript (no frameworks, no dependencies)

## 🚀 Getting Started

### Run Locally

Just open the file in your browser — no server or installation needed:

```bash
# Clone the repo
git clone https://github.com/puspendugorai-coder/Calculator.git

# Open the file
open index.html
```

### Deploy

This is a static site and can be deployed anywhere:

| Platform | Steps |
|----------|-------|
| **Netlify** | Connect GitHub repo → auto-deploys on every push |
| **GitHub Pages** | Settings → Pages → select branch |
| **Hugging Face** | New Space → Static SDK → upload `index.html` |

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Number input |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last character |
| `Escape` | Clear all |
| `( )` | Parentheses |
| `%` | Percentage |
